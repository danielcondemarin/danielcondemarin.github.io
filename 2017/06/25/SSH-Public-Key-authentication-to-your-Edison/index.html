SSH Public Key authentication to your Edison

Jun 25, 2017

I believe the best developers are the laziest ones whom do more by doing less. That includes not having to type in your password every time you want to login to your Intel Edison.

If you are a software developer working with *nix systems, you might be familiar with SSH Public Key authentication. If you don’t know what that is, I strongly recommend having a read of this article, it is by far one of the best explanations I’ve seen.

In a nutshell, we will be generating a private and a public key. Then we will copy ONLY the public key to the Edison. That way you will be able to login to the Edison automatically as long as you have the private key.

Let’s start:

  • Open a terminal:
  1. Generate private / public key pair via ssh-keygen

ssh-keygen -t rsa
